# Firms: Before & After Scenarios

A compact Vite + React recreation of the firms diagram app, organised into four scenario buttons:

1. Fixed Cost
2. Variable Cost per Unit
3. Market Share
4. Economies of Scale

Each scenario displays a baseline **Before** diagram and an interactive **After** diagram controlled by one slider.

## Run locally

```bash
npm install
npm run dev
```

## Build for Vercel

```bash
npm run build
```

Deploy the project root to Vercel. The framework preset should be detected automatically as Vite.
